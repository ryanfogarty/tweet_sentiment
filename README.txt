PROGRAMMER: Ryan Fogarty

PROJECT: Tweet Sentiment

DESCRIPTION: This program searches twitter for key words given by the user on the command line
and prints back to the user, tweets found that were determined (by Tweet Sentiment API) to be 
positive tweets in nature. API's used: tweepy, Tweet Sentiment. 

FILES: tweet_sentiment.py (file that does everything)

SAMPLE COMMAND: python tweet_sentiment.py Bars in Boston
